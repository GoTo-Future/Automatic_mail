пояснение к файлу конфигурации:
server - сервер почты на которую будут приходить сообщения(для mail.ru - smtp.mail.ru)
login - логин аккаунта с которого будут персылаться сообщения
password - пароль(*) от аккаунта с которого пересылаются сообщения
recipients - список пользователей(по их логинам) кому будут пресылаться сообщения
filepath - путь к папке в которой будут пеоявлятся новые файлы
file_archive - путь к папке в которую будут архивироваться файлы после отправки их всем пользователям
name_sender - будет отображено в в качестве имени отправителя(например можно поставить "РМГ" или название системы для которой необходимо эта программа)

*пароль необходимо создать отдельно(пароль от входа в аккаунт не подойдёт) по пути:  настройки Mail ID -> Безопасность -> пароли для внешних приложений -> добавить
для других почтовых сервисов алгоритм может быть другим
